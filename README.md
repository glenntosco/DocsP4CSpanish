# P4 Customs – Manual de Usuario

## Sistema de Gestión Aduanera

P4 Customs es una solución completa de gestión aduanera para importadores, totalmente integrada con **P4 Warehouse**, nuestro sistema de gestión de almacenes.

Con P4 Customs, los importadores tienen la capacidad de administrar sus almacenes aduaneros y gestionar los contratos aduaneros de principio a fin.

---

## Sobre esta documentación

Los datos contenidos en este manual se incluyen con fines demostrativos, o como ejemplos de prueba para que los usuarios se familiaricen con la funcionalidad del sistema. No representan información real, por lo que nombres, códigos, empresas, importadoras y demás registros son ficticios y no deben interpretarse como datos de clientes, proveedores o transacciones verdaderas. Este contenido permite comprender el flujo de procesos sin riesgo de afectar datos reales.
