# Datos Maestros

Los datos maestros son la información central y fundamental que sirve como base para todas las operaciones del sistema.

Esta sección abarca los registros principales que deben configurarse antes de procesar cualquier transacción aduanera:

| Sección | Descripción |
|---|---|
| [Importadores](importadores.md) | Empresas que importan mercancía a través del sistema |
| [Almacenes](almacenes.md) | Almacenes aduaneros gestionados dentro de P4 Customs |
| [Clientes](clientes.md) | Clientes finales que reciben la mercancía |
| [Flete](flete.md) | Proveedores de flete y detalles de transporte |
| [Productos](productos.md) | Catálogo de productos compartido con P4 Warehouse |

Todos los datos maestros configurados en P4 Customs se sincronizan automáticamente con **P4 Warehouse**, garantizando consistencia entre ambos sistemas.
